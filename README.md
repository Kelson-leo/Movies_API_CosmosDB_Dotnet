# MoviesAPI - .NET 10 & Azure Cosmos DB CRUD

A professional-grade RESTful API built with **.NET 10** and **Azure Cosmos DB**, focused on demonstrating modern backend development patterns, asynchronous programming, and containerization.

## 🚀 Features

- **Full CRUD Operations**: Create, Read, Update, and Delete movies.
- **Asynchronous Pattern**: Fully non-blocking I/O using `async/await` across all layers.
- **NoSQL Integration**: Powered by Azure Cosmos DB (Linux Emulator) with custom Partition Key mapping.
- **Containerized Environment**: Ready for development with Docker and Docker Compose.
- **Security First**: Development credentials managed via .NET User Secrets to prevent sensitive data leaks.

## 🛠️ Tech Stack

- **Framework**: .NET 10 (ASP.NET Core)
- **Database**: Azure Cosmos DB (Linux Emulator)
- **ORM**: Entity Framework Core
- **Tooling**: Docker, Bruno (API Testing), Git

## 📋 Prerequisites

- **Docker & Docker Compose**: Installed and running on your system.
- **.NET 10 SDK**: Required for building and running the application.
